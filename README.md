## ‚ú® Core Features

Tackleit is packed with features to make job hunting efficient and effective:

-   **ü§ñ AI-Powered Recommendations**: Utilizes Google's Gemini AI to analyze your profile and generate highly relevant job recommendations.
-   **üí∏ Free vs. Pro Tiers**: A flexible two-tier system.
    -   **Free Plan**: Get monthly job recommendations.
    -   **Pro Plan**: Upgrade to receive weekly recommendations and unlock premium features.
-   **üí≥ Secure Payments**: Seamless and secure subscription management powered by **Razorpay**.
-   **üë§ Unified Authentication**: Easy sign-up and login using either email/password or **Google Sign-In**.
-   **üîí OTP Verification**: Secure email OTP verification for new sign-ups to prevent spam and ensure account validity.
-   **‚öôÔ∏è Detailed User Preferences**: A comprehensive preferences page where you can specify desired roles, tech stack, experience, location, salary, and more to fine-tune your recommendations.
-   **üìÑ Google Sheets Export (Pro Feature)**: Pro users can instantly export their curated job lists to Google Sheets for easy tracking and management.
-   **üì¨ User & Admin Notifications**:
    -   Automated email confirmations for Pro users when they update their preferences.
    -   A robust contact form that forwards user inquiries directly to the admin's email.

## ‚öôÔ∏è Getting Started: Local Setup

Follow these steps to set up and run the project on your local machine.

### Prerequisites

-   **Node.js**: v20.x or later
-   **Python**: v3.9 or later
-   **MongoDB Atlas**: A free cluster is sufficient.
-   **Git**

### 1. Clone the Repository

```bash
git clone https://github.com/saipraneethkotyada/Tackleit.git
cd Tackleit
```

### 2. Backend Setup

```bash
# Navigate to the backend directory
cd backend

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows, use `venv\Scripts\activate`

# Install dependencies
pip install -r requirements.txt

# Create and configure the environment file
cp .env.example .env
```

**Configure your `.env` file** with the following variables:

```env
# backend/.env
MONGO_URI="your_mongodb_connection_string"
DB_NAME="your_database_name"
SECRET_KEY="a_very_strong_random_secret_key"
ALGORITHM="HS256"
ACCESS_TOKEN_EXPIRE_MINUTES=60

# Google Gemini AI
GEMINI_API_KEY="your_gemini_api_key"

# Google OAuth (for Sign-In)
GOOGLE_CLIENT_ID="your_google_client_id"
GOOGLE_CLIENT_SECRET="your_google_client_secret"
REDIRECT_URI="http://localhost:8000/auth/google/callback"

# Razorpay
RAZORPAY_KEY_ID="your_razorpay_key_id"
RAZORPAY_KEY_SECRET="your_razorpay_key_secret"

# SMTP for Email (e.g., Gmail with App Password)
SMTP_SERVER="smtp.gmail.com"
SMTP_PORT=587
SMTP_USERNAME="your_email@gmail.com"
SMTP_PASSWORD="your_gmail_app_password"
SENDER_EMAIL="your_email@gmail.com"
SENDER_NAME="Tackleit"

# Admin/Contact Form
ADMIN_EMAIL="your_admin_email@example.com"
```

### 3. Frontend Setup

```bash
# Navigate to the frontend directory
cd ../frontend

# Install dependencies
npm install

# Create and configure the environment file
cp .env.local.example .env.local
```

**Configure your `.env.local` file** with the following variables:

```env
# frontend/.env.local
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_RAZORPAY_KEY_ID="your_razorpay_key_id"
NEXT_PUBLIC_GOOGLE_CLIENT_ID="your_google_client_id"
```

### 4. Running the Application

1.  **Start the Backend Server**:
    ```bash
    cd backend
    uvicorn main:app --reload
    # The backend will run on http://localhost:8000
    ```

2.  **Start the Frontend Server**:
    Open a new terminal window.
    ```bash
    cd frontend
    npm run dev
    # The frontend will be available at http://localhost:3000
    ```
